# 党建

## 实现“大”目标
加强党对学校工作的全面领导，对标党的教育方针，通过创新育人理念，加强思想道德教育和实践行动的有效衔接，构建全过程育人体系，开展“知、情、意、行”相互协同及灵活多样的思想政治教育，深入落实立德树人根本任务，努力实现为党育人为国育才的大目标。

## 做好“大”教育
聚焦每位学生成长，为做“大”教育增值:
从每一个学生个体的发展出发，坚持因材施教，让每个人都努力成长为最好的自己，真真实现全方位育人、全程育人。以劳动课程、心理课程、国防课程、爱国主义教育课程、毕业课程、综合任务课程、体育综合素养提升课程、少先队自主社团活动、德育实践活动、人生大课等丰富多彩的形式为载体，激发学生内驱力，让学生成为学习的主人。

不断提升教师教育力，为做“大”教育赋能:
加强党员队伍建设，引导教师做“经师”与“人师”的统一者，使基层党支部成为学校教书育人的坚强战斗堡垒。聚焦教师，开展有针对性的专业培训，强化把握正确政治方向，从党小组、党支部、党委等不同层面出发，有设计地构建人才梯队，打造一支胸怀广、视野阔、格局大、站位高的“四有”教师队伍，培育好每一位“大”先生。

树立先锋楷模，为做“大”教育引路:
以党支部为单位，坚持做好党员承诺践诺工作，每学期设立工作目标，并在日常工作中落实落地，通过党支部、党小组的日常交流，民主生活会、组织生活会等多渠道进行评价。聚焦党员先锋岗，将党员在教育教学中的优秀案例、事迹与大家分享，充分发挥党员先锋模范作用，争做引领者。

## 建设“家”文化
倡导“家文化”建设，践行“雁阵精神”:
建强党支部战斗堡垒，让每一个党小组、每一个党支部都成为全体教职员工的家，树立起主人翁意识。学校关心每一位教职员工的成长，在工作生活上支持成就大家，大家齐心协力共建学校“大家庭”。
组织教职工开展丰富多彩活动，建好工会小家，缓解教职工身心压力，提升精神状态，以最饱满的热情投入到工作生活中。营建“家”文化，建强建实各级党组织，党委、党支部、团支部、工会，校区、校中校、班组群都是老师们的家。倡导和建设良好家风，把中关村三小全体教职工凝聚在一起。

通过“双培养”工作，全面提高党员素质，完善党员结构:
努力打造一支高素质的党员队伍，发挥党员在教育教学中的示范引领作用，努力把业务骨干培养成党员，把党员培养成业务骨干。不断努力扩大党员在广大教职工中的吸引力、凝聚力，积极发挥榜样作用，为中关村第三小学教育集团绿色健康、可持续发展做出贡献。


# 民族团结教育

## 铸牢中华民族共同体意识
2008年，中关村三小启动民族团结教育主题活动，2009年56个民族的小伙伴齐聚北京，向新中国成立60周年献礼。此后，学校积极开展回访活动深化教育成果。组织学生回访曾经参与交流活动的民族地区学校，与各民族学生共上一堂课、共升一面旗、共唱一首歌，分享彼此的学习生活与成长变化；教师回访参与过培训研讨的民族地区教育机构，了解当地教育教学的改进情况，并给予进一步的指导和支持。为持续推进中华民族共同体意识教育注入了新动力。同时，学校将民族团结教育与多学科教学融合，开展跨学科活动。2014年，开展“族·迹”主题教育活动为微项目研究，2023年开发“石榴籽”综合任务课程。2024年开始，学校少先队大队带领少先队员开展民族友谊小使者“石榴籽代言人”活动，300多个家庭报名参加，走进各民族地区，与各民族伙伴一起学习一起成长。
2024年9月29日，在新中国成立75周年之际，中关村第三小学举办了一场别开生面的艺术思政课——《好大的家》。这堂课将艺术与思政教育深度融合，通过丰富多彩的教学内容和形式，引导学生深刻理解五十六个民族凝聚在一起，铸牢中华民族共同体意识，树立共同的奋斗目标，推进中国式现代化，实现中华民族伟大复兴。
